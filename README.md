# Bienvenue sur le projet d'audit MacOS

### Comment exécuter le script 

1. Cloner ou télécharger le repo
2. Dans `macos_hardening`
```bash
cd macos_hardening
```
2. Run this command :
```bash
chmod u+x ./puppy.sh
./puppy.sh
```
